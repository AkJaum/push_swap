**Description**
The push swap project order nums with limited operations, are they:
- Swap
Swap the first two elements of a stack
- Push
Takes the first element of a stack and put it to another
- Rotate
Shift up all elements of a stack by 1 and the first element becomes the last one
- Reverse rotate
Shift down all elemtents of a stack by 1 and the last element becomes the first one. 
This project also focus in optimizing the project for the minimal number of operations to order
a stack.

**Instructions**
You can simply compile the project using the command make, and then, run the program and givingit args to order.
Example:
./push_swap "5 8 2 6 0"
If all the arguments are valid, it will print the operations he used to order the entire stack, and using wc -l at the end of the command line, you will see the number of steps the program used.
